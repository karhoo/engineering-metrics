Introduction
------------
This library provides a wrapper interface a set of data sources. The idea is to enable the production of Enginering Metrics for your team by analysis of metrics from across the business. It works with Python 3.6 (and above?)

We currently provide wrappers for:

- `Jira <https://jira.readthedocs.io/en/master/>`_.
